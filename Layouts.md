# Layouts in Android

Layout basically refers to the arrangement of elements on a page these elements are likely to be images

## Types of Layouts
- `LinearLayout`
- `RelativeLayout`
- `ConstraintLayout`
- `TableLayout`
- `FrameLayout`
- `ListView`
- `GridView`
- `AbsoluteLayout`
- `WebView`
- `ScrollView`
